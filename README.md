# Research Agent for Obsidian

Research Agent is a conversational research plugin for Obsidian. It connects to
Deep Research Cloud at `https://research.obclaude.com`.

## Install with BRAT

1. In Obsidian, install and enable `BRAT` from Community plugins.
2. Run `BRAT: Add a beta plugin for testing`.
3. Paste this repository URL:

```text
https://github.com/0126-hash/obsidian-research-agent
```

4. Enable `Research Agent` in Community plugins.

New beta users need an invite code from the service administrator.
